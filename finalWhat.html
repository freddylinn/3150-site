What Exactly is Universal Design?

Principles of Universal Design

The Centre for Excellence in Universal Design defines universal design as "the design and composition of an environment so that it can be accessed, understood and used to the greatest extent possible by all people regardless of their age, size, ability or disability." This serves as a strong foundation for what we consider universal design to be as a concept. These pages in particular will focus on universal design on the web, but it can be extended to every part of everyday life. The core idea of designing products that are immediately accessible to all users can guide designers toward a common goal. There are seven commonly used principles that can help achieve this.

  1. Equitable Use
  2. Flexibility in Use
  3. Simple and Intuitive Use
  4. Perceptible Information
  5. Tolerance for Error
  6. Low Physical Effort
  7. Size and Space for Approach and Use

An excellent reference for universal design is the book The Design of Everyday Things by Don Norman. The book has multiple iterations, but each one encompasses imperative concepts for design that will automatically make sense without the user having to even think about it. These include affordances, which are aspects of the design that are perceivable possibilities by the user. Constraints are limits put on what the user can do to prevent user error. Feedback is the concept of a product responding in some way when the user interacts with it correctly or incorrectly. All three of these can be applied to website design, and Norman covers many more topics in his book.

Tools for Digital Universal Design

Assistive technologies have revolutionized how users with disabilities can interact with their computers. These tools offer a wide range of possibilities when set up correctly. Below is a list of some assistive technologies or concepts that help improve the digital experience for users with disabilities.

Visual Hearing Motor
Screen Readers Video Captions Keyboard Navigation
Adjustable Font Size Bone Conduction Headphones Mouth Stick
Colorblind Color Schemes Haptic Feedback Vests Eye Tracker
